These files were used to create a website hosted on Google Cloud Platform. GCP's platform allows for password protection and cheap hosting.

First, submissions are uploaded using a Google Form on the website (https://gcmteachinghub.commons.gc.cuny.edu/). The results of this form are stored on a Google Sheet. 

The processing form/website automatically draws from this sheet and helps to expedite the process. It can apply OCR, upload files to the WordPress site, and create a draft of a page based on the form. This takes a very tedious task and massively reduces the amount of time it takes. More importantly, it reduces the amount of re-learning time by simplifying the process (we receive submissions very sporadically).
